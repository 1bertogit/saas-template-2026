# Arquitetura

- Next.js App Router com rotas para marketing, auth e dashboard.
- Clerk para autenticação com middleware de proteção.
- Stripe para pagamentos e webhooks de assinatura.
- Neon + Drizzle para persistência.
- Cloudflare R2 para storage de arquivos.
- Resend para emails.
- PostHog para analytics.
- OpenRouter para IA via OpenAI SDK.
